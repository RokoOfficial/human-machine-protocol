# Welcome to the Human Machine Protocol (HMP) Documentation

**Official Website:** [openroko.org](https://www.openroko.org)

---

This documentation provides a comprehensive guide to the Human Machine Protocol (HMP), a powerful and deterministic protocol for orchestrating complex workflows between humans, machines, and Large Language Models (LLMs).

Whether you are a developer looking to integrate HMP into your applications, an architect designing intelligent systems, or a user leveraging HMP-powered agents, this documentation contains the information you need to succeed.

## Documentation Sections

*   **[Introduction](./01-introduction.md):** A high-level overview of HMP, its purpose, and its core concepts.
*   **[Getting Started](./02-getting-started.md):** A step-by-step tutorial to get you up and running with HMP.
*   **[Technical Specification](./spec/):** A detailed reference of the HMP language, including syntax, variables, tools, and control flow.
